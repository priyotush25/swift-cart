1. Null এবং Undefined এর পার্থক্য

Null: প্রোগ্রামার নিজেই assign করে “value নেই” বোঝাতে।

Undefined: variable declare করা হয়েছে কিন্তু কোনো value assign হয়নি।

2. map() ব্যবহার এবং forEach থেকে পার্থক্য

map(): array এর প্রতিটি element modify করে নতুন array return করে।

forEach(): প্রতিটি element এ কাজ করে কিন্তু কিছু return করে না।

3. == এবং === পার্থক্য

== : value check করে, type ignore করতে পারে।

=== : value এবং type দুইটিই check করে।

4. async/await এর গুরুত্ব

asynchronous operation সহজে handle করার জন্য।

await দিয়ে Promise resolve হওয়া পর্যন্ত code pause করে।

cleaner ও sequential code লেখা যায়।

5. Scope এর ধারণা

Global Scope: সব জায়গা থেকে access possible।

Function Scope: শুধু function এর ভিতরে access possible।

Block Scope: শুধু block {} এর ভিতরে access possible (let ও const এর জন্য)।
